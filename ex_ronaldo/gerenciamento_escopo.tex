\subsubsection{Escopo}
O escopo deste projeto foi desenvolvido com base na análise das necessidades
do cliente e deverá contemplar os seguintes itens:

\begin{itemize}
\item{Plano de ação:
\newline
     Serão levantadas as causas da atual necessidade do cliente por meio de
     um diagrama de causa e efeito (também conhecido como 7M), a partir
     do qual um plano de ação será elaborado contendo orientações para a
     correção destas causas.
}

\item{Auxílio para seleção de produtos/serviços:
\newline
     Baseando-se em pesquisas de mercado, a \Empresa\  orientará o
     cliente na seleção e aquisição/contratação de produtos (softwares de
     gestão de impressões) e serviços (instalação, configuração de
     programas e manutenção de equipamentos) adequados às necessidades do
     negócio.
}

\end{itemize}


\newpage
\subsubsection{Fora do Escopo}
O projeto não contemplará os seguintes itens:

\begin{itemize}
\item{Fornecimento de equipamentos:
\newline
     A aquisição das impressoras, bem como os materiais necessários para
     seu funcionamento, manutenção são de responsabilidade da empresa cliente.
}

\item{Logística dos insumos:
\newline
    O estoque de insumos é de responsabilidade da empresa cliente.
}

\item{Suporte e treinamento técnicos aos usuários:
\newline
      Tais serviços deverão ser prestados pela assistência técnica da
      empresa responsável pelo desenvolvimento das tecnologias utilizadas
      nos processos de impressão.
}

\item{Instalação e configuração de programas:
\newline
      A \Empresa\  apenas orientará o cliente na seleção e aquisição dos \softwares. A instalação
      e a configuração destes são de responsabilidade de uma empresa
      terceirizada contratada para este fim.
}

\end{itemize}
