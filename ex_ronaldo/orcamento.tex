O orçamento foi desenvolvido com base na agregação dos custos estimados
de atividades individuais da EAP, estabelecendo uma linha base dos custos
totais para a medição do valor do projeto.

\newpage
\begin{enumerate}
\item{Atividades:
\newline
  (A1 + A2) + (A1 + A2 + A3) + (A1 + A2 + A3 + A4 + A5)
}

\item{Pacotes de Trabalho:
\newline
  PT1 = A1 + A2 = R\$ 35,00 + R\$ 300,00 => PT1 = R\$ 335,00\\
  PT2 = A1 + A2 + A3 = R\$ 300,00 + R\$ 40,00 + R\$ 40,00 => PT2 = R\$ 380,00
  PT3 = A1 + A2 + A3 + A4 + A5 = R\$ 60,00 + R\$ 50,00 + R\$ 45,00 + R\$ 700,00 + R\$ 1.350,00 => PT3 = R\$ 2.205,00 
}

\item{Contas de Controle:
\newline
  CC1 = PT1 + PT2 + PT3 = R\$ 335,00 + R\$ 380,00 + R\$ 2.205,00 => \\ CC1 = R\$ 2.920,00
}

\item{Projeto:
\newline
  Projeto = CC1 = R\$ 2.920,00
}

\item{Reserva de Contingência: R\$ 0,00 (todos os riscos deste projeto foram transferidos para o cliente).}

\item{Linha de Base de Custos:
\newline
  Projeto + Reserva de Contingência = R\$ 2.920,00 + 0 
}

\item{Reserva de Gerenciamento:
\newline
  Projeto * 10\% = R\$ 2.920,00 * 0,1 = R\$ 292,00
}

\item{Orçamento Final:
\newline
  Linha de Base + Reserva de Gerenciamento = R\$ 2.920,00 + R\$ 292,00 = \\ R\$ 3.212,00
}

\end{enumerate}
